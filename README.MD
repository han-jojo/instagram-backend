# Instagram 클론

nodeJS 를 이용하여 인스타그램을 클론코딩 합니다.

## 사용된 기술

- 프론트 엔드 : Apollo, React, Styled Components
- 백엔드 : Apollo Server, GraphQL, Prisma
- 앱 : iOS & Android: React Native, Expo, React Native CLI, React Navigaion, React Native Web
